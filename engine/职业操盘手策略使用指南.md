## 🆕 新增策略详解

### 1. 情绪周期策略 (`sentiment_cycle`)

**策略类型**: 游资核心策略

**核心理念**:
- 情绪冰点（恐惧）→ 试探性买入
- 情绪回暖 → 加仓
- 情绪高潮 → 减仓
- 情绪退潮 → 空仓

**使用场景**:
- 市场波动剧烈时
- 需要判断市场情绪转折点
- 短线交易，捕捉情绪波动

**参数说明**:
```python
SentimentCycleStrategy(
    fear_period=20,        # 恐惧周期
    greed_period=10,        # 贪婪周期
    fear_threshold=0.3,     # 恐惧阈值
    greed_threshold=0.8     # 贪婪阈值
)
```

**优势**:
- 符合人性弱点，利用情绪波动获利
- 适合短线操作
- 有明确的买入卖出时机

---

### 2. 行业轮动策略 (`sector_rotation`)

**策略类型**: 机构核心策略

**核心理念**:
- 低估值 → 高景气 → 政策风口
- 模拟行业轮动周期

**使用场景**:
- 行业轮动行情
- 捕捉热点切换
- 中长线配置

**参数说明**:
```python
SectorRotationStrategy(
    lookback=60,            # 回看周期
    momentum_window=20,     # 动量窗口
    value_threshold=0.3,    # 价值阈值
    momentum_threshold=0.1  # 动量阈值
)
```

**优势**:
- 机构常用策略，成熟可靠
- 价值与动量结合
- 适合捕捉行业轮动机会

---

### 3. 景气度投资策略 (`prosperity`)

**策略类型**: 私募核心策略

**核心理念**:
- 行业高增长、订单饱满、产品涨价
- 核心：景气 + 低估值

**使用场景**:
- 成长股投资
- 高景气行业配置
- 中线布局

**参数说明**:
```python
ProsperityInvestmentStrategy(
    growth_window=30,           # 增长窗口
    min_growth_rate=0.15,       # 最小增长率
    volume_confirm=True,        # 成交量确认
    volume_threshold=1.5        # 成交量阈值
)
```

**优势**:
- 私募最主流策略
- 成长股投资利器
- 量价配合，信号可靠

---

### 4. 波段操作策略 (`band_operation`)

**策略类型**: 私募核心策略

**核心理念**:
- 低吸高抛，不追高
- 回调买，突破卖
- 不长期死拿、不频繁交易
- 一年操作不超过10次

**使用场景**:
- 震荡市行情
- 中线波段交易
- 高抛低吸

**参数说明**:
```python
BandOperationStrategy(
    band_window=20,           # 波段窗口
    buy_band_pct=0.3,        # 买入区间
    sell_band_pct=0.7,       # 卖出区间
    min_hold_days=5,          # 最小持仓天数
    max_hold_days=60          # 最大持仓天数
)
```

**优势**:
- 符合"低吸高抛"原则
- 有明确的持仓周期限制
- 避免频繁交易

---

### 5. 价值投资策略 (`value_investment`)

**策略类型**: 机构核心策略

**核心理念**:
- 现金流好、ROE高、分红稳定
- 行业龙头
- 买了拿几年，不折腾

**使用场景**:
- 长线投资
- 价值股配置
- 稳健型投资者

**参数说明**:
```python
ValueInvestmentStrategy(
    rebalance_days=250,       # 再平衡周期
    min_roe=0.15,            # 最小ROE
    max_pe=30,               # 最大PE
    min_dividend=0.02,       # 最小分红率
    trend_window=60           # 趋势窗口
)
```

**优势**:
- 机构最常用策略
- 适合长期持有
- 风险较低，收益稳定

---

### 6. 龙头战法 (`dragon_head`)

**策略类型**: 游资核心策略

**核心理念**:
- 只做最强龙头
- 板块最强股票
- 涨停突破确认买入
- 快进快出，不隔夜

**使用场景**:
- 热点题材炒作
- 短线暴利机会
- 追击龙头股

**参数说明**:
```python
DragonHeadStrategy(
    strength_window=20,           # 强势窗口
    limit_up_threshold=0.095,     # 涨停阈值
    breakout_confirm=True,         # 突破确认
    hold_max_days=3,               # 最大持仓天数
    stop_loss_pct=0.03            # 止损比例
)
```

**优势**:
- 游资第一策略
- 高风险高收益
- 严格的止损机制

⚠️ **注意**: 此策略风险较高，建议谨慎使用！

## 💰 资产配置管理器

### AllocationManager

大师兄体系 - 顶级操盘手资产配置:

**目标配置**:
- 🥇 30%-50% 稳健仓 - 高分红、稳现金流、弱周期
  - 示例: 长江电力、中国神华、贵州茅台、中国移动

- 🥈 10%-15% 进攻仓 - 高景气、国产替代、硬科技
  - 示例: 半导体、AI、新能源车、算力

- 🥉 20%-30% 指数仓 - 沪深300、中证500宽基
  - 示例: 510300、510500

- 💰 10%-20% 现金仓 - 预留现金，大跌抄底

**风控原则**:
- 永远不满仓、不加杠杆
- 单只股票不超过总资金 10%
- 止损必须执行：10%-20% 无条件砍
- 不追高、不炒题材、不赌消息
- 不频繁交易、一年操作不超过 10 次
