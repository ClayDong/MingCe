### 3. 核心功能说明

#### FeishuNotifier
```python
# 发送 Markdown 消息
notifier.send_markdown("Hello")

# 发送日报
notifier.send_daily_report(report_content)

# 发送告警
notifier.send_alert("标题", "内容", "error")
```

#### SystemMonitor
```python
# 检查系统健康
health = monitor.check_system_health()

# 获取状态摘要
summary = monitor.get_status_summary()
```
